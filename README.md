# Prediksi Produksi Bawang Merah Sulawesi Tengah

Proyek machine learning untuk memprediksi produksi bawang merah di Sulawesi Tengah menggunakan algoritma Random Forest Regression.

## Dataset

Dataset berisi faktor-faktor yang mempengaruhi produksi bawang merah, antara lain:

- Tahun
- Kecamatan
- Curah Hujan
- Suhu
- Kelembapan
- Luas Panen
- Penggunaan Pupuk
- Serangan Hama
- Jumlah Petani

## Menjalankan Program

Install library:

```bash
pip install -r requirements.txt
```

Jalankan program:

```bash
python Program_dataset_bawang.py
```
